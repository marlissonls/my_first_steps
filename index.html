Em um software editor de texto para editar documentos, livros, jornais e etc. é possível usar o teclado e as opções do menu que o próprio software oferece para editar qualquer tipo de texto e imagens com toda a sorte de formatações possíveis, por exemplo: In Design, Microsoft Word, FocusWriter, entre outros.

Para escrever publiações em sites, no entanto, é necessário escrever e editar páginas usando a linguagem suportada pelos navegadores de internet. Essa linguagem é chamada de HTML - HyperText Markup Language. Os hipertextos são conjuntos de elementos conectados. Esses elementos podem ser palavras, imagens, vídeos, documentos, etc.

Em qualquer página na internet é possível observar diferentes distribuições e tamanhos para títulos, parágrafos, imagens e qualquer outro elemento. Essa formatação é estabelecida através do HTML, o qual é estruturado em conjuntos de tags, que são blocos de construção das páginas responsáveis pela hierarquização, marcação e estrutura dos conteúdos, dividindo entre seções, parágrafos, cabeçalhos, entre outros. A propósito, este texto é um parágrafo e foram utilizadas as tags <p> e </p> para configurá-lo desta forma.

Nesta publicação, iremos demonstrar o uso das tags básicas em uma página estruturada em HTML.

As tags de abertura e fechamento <html> e </html> possibilitam que as informações sejam lidas nos browsers e englobam tudo o que é renderizado nas páginas da internet.

As tags <head> e </head> recebem informações básicas da página como o título da página, links externos e metadados.

A tag <meta charset="utf-8"/> indica o formato de codificação definido pelos caracteres que serão usados em uma determinada página. Nesta página o codificação de caractere usada é o UTF-8.

As tags <title> e </title> servem para definir o título da página. Nesse exemplo o título da página é Alpha Edtech, e está exibido na aba da página na barra superior do browser.

As tags <body> e </body> comportam todas as informações que farão parte do corpo da página. Inclusive este conteúdo que você está lendo agora.

As tags <sup> e </sup> configuram um texto sobrescrito, por exemplo: texto comumtexto sobrescrito.

As tags <sub> e </sub> configuram um texto subescrito, por exemplo: texto comumtexto subescrito.
